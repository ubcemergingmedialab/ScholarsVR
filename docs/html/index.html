ScholarsVR

Project Information

This project is a collaborative effort between UBC Emerging Media Lab, Centre for Digital Media (CDM), and SFU School of Interactive Arts and Technology (SIAT). This goal of this project is to create a gamified VR experience to present scholarly articles to visitors at the Koerner's Library at UBC.

Work on the project began in October 2019 and is expected to continue till December 2019.
